# T Group Intra (Sugestão 1 • MVP local JSON)

## Rodar
npm install
npm run dev

## Produção (Vercel)
- Crie repo GitHub, push o código

- data/config.json: almocoUrl, nfLink, beneficios, manuais, capacitar
- data/events.json: eventos do calendário
